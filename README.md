# 🧰 Scanlation Forge

A toolkit for scanlation teams looking to automate the manga production process, from raw hunting to RD and Type. 

Have a script that can save hours of work? **Contributions are highly encouraged!**

This repository is built **by scanlators, for scanlators**. Feel free to use, fork, and contribute.

## Toolkit Overview

The repository is organized into two primary categories to cover the full production pipeline:

### 🌐 Web Automation
Tools designed to streamline content retrieval and archiving from various manga hosting platforms.

![Status](https://img.shields.io/badge/Status-Under_Construction-yellow)

### 🎨 Image Processing
Scripts and actions to accelerate cleaning, redrawing, and typesetting workflows.

![Status](https://img.shields.io/badge/Status-Under_Construction-yellow)

## Ethical Disclaimer
These tools are provided for the archival and preservation of manga. I encourage users to support official releases whenever possible.
